# Simon Says Game

This is a simple implementation of the classic "Simon Says" game in Python. The game challenges players to remember and repeat a sequence of colors. Each round, the sequence gets longer, and the player must keep up to win.

This is a simple implementation of the classic "Simon Says" game in Python. The game challenges players to remember and repeat a sequence of colors. Each round, the sequence gets longer, and the player must keep up to win.

## How to Play

1. Run the script to start the game.
2. The game will display a sequence of colors (Red, Green, Blue, Yellow).
3. After the sequence is shown, you will be prompted to enter the colors in the correct
4. order. You can enter the colors separated by spaces (e.g., "Red Green Blue").
5. If you enter the correct sequence, you will move on to the next round, where the sequence will be longer.
6. If you enter the wrong sequence, the game will end, and your score will be
7. displayed.
8. Have fun playing and see how far you can go!

- Python 3.x
- No additional libraries are required.
